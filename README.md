## Library
- A library system Frontend
<br /><br />
- Built using 
    + **`HTML5`**
    + **`CSS`**
      - flexbox
      - grid
    + **`JavaScript`**
<br /><br />
- It has the functionallity to
  - add book
  - remove book
  - change book state (read it or not)

- You can access a live preview [**`here`** :link:](https://mhmad-alaa.github.io/library/)
