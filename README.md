## Bookset
- A library system Frontend
<br /><br />
- Built with
    - **`HTML5`**
    - **`CSS`**
    - **`JavaScript`**
<br /><br />
- It has the functionallity to
  - Add book
  - Remove book
  - Change book status (readed or not)
<br /><br />
